# 🆘 SOS SURVIVE — Офлайн-ассистент выживания

**Полностью офлайн-приложение для экстремальных ситуаций, походов и путешествий.**

---

## 🌟 Возможности

| Фича | Описание |
|------|----------|
| 🌍 **2 языка** | Русский и English — переключай в один клик |
| 📴 **Полностью офлайн** | Работает без интернета после первого открытия |
| 📱 **PWA-приложение** | Устанавливается на телефон как нативное |
| 🎤 **Голосовой ввод** | Скажи "Вода" — откроется раздел |
| 🌙☀️ **Тёмная/светлая тема** | Выбирай удобную |
| 🆘 **SOS-режим** | GPS, сигналы, фонарик с SOS-миганием |
| 🧠 **Психология выживания** | Советы для сохранения спокойствия |
| 🗺️ **Карты** | Ссылка на OpenStreetMap + Organic Maps |

## 🚀 Быстрый старт

### 1. Запуск на компьютере

```bash
# Установи Node.js с https://nodejs.org
npm install -g serve

# Перейди в папку проекта
cd sos-survive

# Запусти сервер
serve .

# Открой http://localhost:3000
