# 🆘 SOS SURVIVE — Офлайн-ассистент выживания

## Быстрый старт

### 1. Установка (Windows/Mac/Linux)
```bash
# Установи Node.js с https://nodejs.org
npm install -g serve
cd sos-survive
serve .
```

### 2. Открытие
Открой `http://localhost:3000` в браузере.

### 3. Установка на телефон
- **Android Chrome**: Меню → "Добавить на главный экран"
- **iOS Safari**: Поделиться → "На экран «Домой»"

### 4. Офлайн-режим
После первого открытия приложение работает без интернета. Service Worker кэширует все файлы.
